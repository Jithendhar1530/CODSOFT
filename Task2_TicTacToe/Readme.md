# 🎮 Tic-Tac-Toe AI (Task 2 - CODSOFT AI Internship)

## 📌 Project Description
This project is a Tic-Tac-Toe game developed using Java where a human player plays against an AI.

The AI uses the **Minimax algorithm** to make optimal moves, making it unbeatable.

This project helps in understanding:
- Game theory basics
- Minimax algorithm
- Decision making in AI
- Search algorithms

---

## 🚀 Features
- Human vs AI gameplay
- AI makes intelligent moves
- Unbeatable AI using Minimax algorithm
- Simple console-based interface
- Detects win, loss, and draw conditions

## 💡 How It Works
- Player uses **X**
- AI uses **O**
- Player enters row and column (0–2)
- AI calculates the best move using Minimax
- Game continues until:
  - Player wins
  - AI wins
  - Draw
