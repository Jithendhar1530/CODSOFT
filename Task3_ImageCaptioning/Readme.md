# 📸 Image Captioning AI (Task 3 - CODSOFT AI Internship)

## 📌 Project Description
This project implements an Image Captioning system that generates textual descriptions for input images.

It combines:
- Computer Vision (to understand images)
- Natural Language Processing (to generate captions)

A pretrained transformer-based model is used to generate captions automatically.

---

## 🚀 Features
- Generates captions for any input image
- Uses pretrained deep learning model
- Simple and easy to run
- No need for training the model
- Supports different types of images

---

## 🛠️ Technologies Used
- Python
- Transformers (Hugging Face)
- Torch
- Pillow (PIL)
