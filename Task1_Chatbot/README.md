# 🤖 Rule-Based Chatbot (Task 1 - CODSOFT AI Internship)

## 📌 Project Description
This project is a simple rule-based chatbot developed using Java.  
The chatbot responds to user inputs based on predefined rules using if-else statements.

This project helps in understanding:
- Basic Natural Language Processing (NLP)
- Conversation flow
- Handling user input in Java

---

## 🚀 Features
- Responds to greetings (Hi, Hello)
- Answers simple questions
- Provides help messages
- Interactive console-based chatbot
- Exit option using "bye"

---

## 🛠️ Technologies Used
- Java
- Scanner Class
